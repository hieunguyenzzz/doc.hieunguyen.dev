# Magento Boilerplate
## Get all child of configurable product

```php
foreach($collection as $_product) {
        $logger->info("Here are Parent Product Name".$_product->getName());
        $_children = $_product->getTypeInstance()->getUsedProducts($_product);
        foreach ($_children as $child){
            $logger->info("Here are your child Product Ids ".$child->getID());
        }
    }
```

## Use resource model to query to database directly

```php
public function __construct(
        \Magento\Framework\App\ResourceConnection $resourceConnection
    ) {
        $this->resourceConnection = $resourceConnection;
    }

protected function query()
{
    if (!$this->connection) {
        $connection = $this->resourceConnection->getConnection('core_write');
    }

    $connection->select();
}
```
